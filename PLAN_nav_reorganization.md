## 5번: 부동산 투자 기능 추가

### 방향
사이트 브랜드("Stock Insight AI")를 유지하면서 부동산을 **투자 도구** 및 **상장 리츠 분석**으로 자연스럽게 흡수.
서브타이틀/설명 문구를 *"주식 · 부동산 통합 투자 분석"* 으로 확장.

### A. 부동산 계산기 — `/tools` 허브 확장

기존 계산기 섹션 아래 "🏠 부동산 투자" 소제목으로 4개 추가:

| 계산기 | URL | 설명 |
|--------|-----|------|
| 임대수익률 계산기 | `/rental-yield-calculator` | 매수가·보증금·월세 → 순수익률 |
| 전월세 전환 계산기 | `/jeonse-calculator` | 보증금 ↔ 월세 전환, 전환율 비교 |
| 대출 상환 계산기 | `/mortgage-calculator` | 원리금균등·원금균등 월 상환액 |
| 부동산 vs 주식 비교 | `/realestate-vs-stock` | 동일 자금 투자 시 장기 수익 비교 ← 킬러 콘텐츠 |

> **부동산 vs 주식 비교**가 핵심. 국내에 유사 도구가 거의 없으며 Stock Insight의 정체성과 완벽히 일치.

#### 작업 목록
1. 4개 HTML 템플릿 생성 (JavaScript 계산, API 불필요)
2. `main.py`에 4개 라우트 추가
3. `templates/tools.html`에 "🏠 부동산 투자" 섹션 추가
4. `templates/base.html` active 조건에 4개 URL 추가

### B. 리츠(REITs) 탐험가 — `/reit-explorer`

국내 상장 리츠는 증권거래소에 상장된 주식이므로 사이트 정체성과 100% 일치.
기존 `/etf-explorer`와 유사한 구조로 구현.

| 항목 | 내용 |
|------|------|
| 데이터 소스 | yfinance (국내 리츠 티커: 맥쿼리인프라 088980.KS, SK리츠 395400.KS 등) |
| 주요 표시 정보 | 배당수익률, 주가, 52주 범위, NAV 대비 할인율 |
| 정렬/필터 | 배당수익률 순, 섹터별(오피스/물류/주거/인프라) |

#### 작업 목록
1. `templates/reit_explorer.html` 생성
2. `main.py`에 `GET /reit-explorer` 라우트 추가
3. 내비게이션에 추가 (ETF 탐험가 인근 배치)
4. `/sitemap.xml`에 URL 추가

## 6번: 뉴스레터 구독 기능 개선

### 현재 문제점

| 문제 | 원인 | 증상 |
|------|------|------|
| 이메일이 실제로 발송되지 않음 | 발송 로직 없음 (저장만 함) | 구독해도 메일 미수신 |
| 구독 데이터 유실 | `static/newsletter_subscribers.json`을 런타임에 생성하나 Render 배포 시 파일시스템 초기화 | 구독자 목록 배포마다 초기화 |
| 파일 기록 실패 가능 | 클라우드 환경에서 `static/` 쓰기 권한 제한 가능 | 500 오류 발생 가능 |

### 개선 방향 (경량 무료 방식 권장)

**옵션 1: Mailchimp 연동 (권장)**
- 무료 플랜: 구독자 500명, 월 1,000건 발송
- Mailchimp API로 구독자 추가 → 캠페인 작성/발송은 Mailchimp 대시보드에서
- 환경변수 `MAILCHIMP_API_KEY`, `MAILCHIMP_LIST_ID` 추가 필요

**옵션 2: EmailOctopus 연동**
- 무료 플랜: 구독자 2,500명, 월 10,000건 발송 (Mailchimp보다 넉넉)
- API 구조 단순, 한국어 지원

**옵션 3: Google Sheets + Gmail (최소 구현)**
- Google Sheets API로 구독자 저장 (데이터 영속성 해결)
- Gmail SMTP로 수동 또는 자동 발송
- 추가 비용 없음, 단 발송 자동화가 번거로움

### 작업 목록 (옵션 1 기준)
1. Mailchimp 계정 생성 및 Audience 리스트 생성
2. `main.py` `/api/newsletter` 엔드포인트 수정
   - JSON 파일 저장 → Mailchimp API 호출로 교체
3. Render 환경변수에 `MAILCHIMP_API_KEY`, `MAILCHIMP_LIST_ID` 등록
4. 주간 발송 콘텐츠 템플릿 설계 (시장 요약, AI 분석 하이라이트)
